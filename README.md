# Gym Management System

A desktop application built in Java that simplifies the management of a sports hall/gym. The application features a clean Graphical User Interface (GUI) and allows administrators to easily track members, trainers, and fitness classes. 

This project was developed to practice Object-Oriented Programming (OOP) concepts, UI design with Java Swing, and file I/O operations.

## Features
* **Member Management:** Register new members with unique IDs and view the active member list.
* **Trainer Allocation:** Add trainers along with their specific specializations (e.g., Fitness, Yoga, Crossfit).
* **Class Scheduling:** Create and monitor fitness classes, including their type and maximum capacity.
* **Data Persistence:** Automatically save and load all records to/from local text files, ensuring data is not lost between sessions.
* **Graphical Interface:** An intuitive, event-driven GUI built entirely with standard Java libraries (Swing/AWT) - no external dependencies required.

## Technologies Used
* **Language:** Java (JDK 8+)
* **GUI Framework:** Java Swing & AWT
* **Data Storage:** File I/O (Plain text parsing)
* **Architecture:** Object-Oriented Design (Inheritance, Encapsulation, Event Listeners)
